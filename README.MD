Running Instructions
- download github
- then download the repo
- download & install python
- then go into the directory ie ~/Docsandsettsings/smugrubber
- and in console do:
  -  ``` python -m SimpleHTTPServer 8000 ``` 
- then open http://127.0.0.1:8000 in your browser
